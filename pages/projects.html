Portfolio

A selection of projects I have worked on

In Progress: Todo App

A mobile application built in React Native maintains daily tasks that a user needs to complete throughout the day.

SmartTalk

A gamified mobile app & web integration designed to assist non-native speakers in English through Automated Speech Recognition (ASR) technology.

Simpson Chatbot

Web application built in Javascript and Speech Synthesis API to communicate with a fictional character named Bart Simpson. Bart will be able to speak to the user by reading what they have inserted into the input box that has been provided already with a list of questions a user can ask him. As well as being able to keep a record of every message and display those messages above the input box.

Sneakers

An e-commerce web application that is built in Javascript, HTML, and CSS. As well as using flexbox to make the application mobile responsive. A user will be able to shop through a selection of shoes to pick a type of shoe, color, and size. If the user wants to purchase that shoe they can do so by using the payment form.
